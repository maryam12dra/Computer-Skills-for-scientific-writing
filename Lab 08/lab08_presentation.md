# 3. TikZ: графика как код

TikZ — мощный пакет для программного создания графики в LaTeX.

```latex
\begin{tikzpicture}
\draw (-1,0) -- (3,10pt) -- (35:3);
\end{tikzpicture}
```

Это позволяет строить рисунки через координаты, команды и стили без внешнего графического редактора.

---

# 3.1 Базовые строительные блоки

В TikZ важны:
- `tikzpicture`;
- команды `\draw`;
- координаты;
- стили линий и цветов.

Пример:

```latex
\draw[-&gt;] (-1,0) -| (3,10pt);
\draw[red] (3,10pt) -- (35:3);
```

---

# 3.2 Линии, кривые и узлы

TikZ поддерживает прямые, кривые Безье и узлы с текстом.

```latex
\node[circle, draw] at (0,0) (a) {A};
\draw[green] (-1,0) to[out=90,in=135] (5,1);
\draw[cyan] (-1,0) .. controls (0,-2) .. (5,1);
```

Это позволяет создавать графы, схемы и подписанные диаграммы.

# 3.3 Построение графиков функций

TikZ умеет строить графики напрямую по формуле.

```latex
\draw[blue, thick] [domain=-2:2, samples=150]
plot (\x, {cos(pi*\x r)});
```

В отчёте рассматривались:
- оси координат;
- сетка;
- подписи функций;
- визуальное оформление кривых.

# 3.4 Циклы и рекурсия

Для сложных рисунков можно использовать `\foreach` и рекурсивные функции.

```latex
\foreach \i in {0,1,2,3} {
  % repeated drawing
}
```

Такой подход особенно полезен для фракталов и повторяющихся структур.
