# 3. Математический режим / Math mode

В LaTeX есть два основных режима для математики:

- **inline** — формула внутри строки текста;
- **display** — формула как отдельный центрированный блок.

Пример:

```latex
A sentence with inline mathematics: $y = mx + c$.

\[
y = mx + c
\]
```

---

# 3.1 Inline math mode

**Inline mode** удобно использовать внутри обычного текста.

Примеры:

```latex
Superscripts $a^{b}$ and subscripts $a_{b}$.
Some mathematics: $y = 2 \sin \theta^{2}$.
```

Что демонстрируется:

- верхние и нижние индексы;
- математические символы внутри абзаца;
- компактная запись формул без разрыва текста.

---

# 3.2 Display mathematics

**Display math** применяется для более крупных формул и интегралов.

```latex
\[
\int_{-\infty}^{+\infty} e^{-x^2} \, dx
\]
```

Также можно использовать окружение `equation`:

```latex
\begin{equation}
\int_{-\infty}^{+\infty} e^{-x^2} \, dx
\end{equation}
```

---

# 3.3 Пакет `amsmath`

Пакет `amsmath` расширяет стандартные возможности набора формул и выравнивания.

```latex
\begin{align*}
Q_{n,k} &= Q_{n-1,k} + Q_{n-1,k-1} + \binom{n}{k}, \\
&\quad \text{for } n,k>0.
\end{align*}
```

Преимущества:

- выравнивание по символу `=`;
- аккуратное оформление многострочных выражений;
- встроенная поддержка математических структур.

---

# 3.4 Шрифты в математическом режиме

В математике разные шрифты обозначают разные типы объектов.

```latex
$\mathrm{A}, \mathit{A}, \mathbf{A}, \mathsf{A}, \mathtt{A}, \mathbb{A}$
```

Использованные команды:

- `\mathrm{}`
- `\mathit{}`
- `\mathbf{}`
- `\mathsf{}`
- `\mathtt{}`
- `\mathbb{}`

---

# 3.5 Дополнительные выравнивания

Для многострочных выражений можно применять окружения `gather`, `multline` и другие.

```latex
\begin{gather}
P(x)=ax^{5}+bx^{4}+cx^{3}+dx^{2}+ex+f\\
x^2+x=10
\end{gather}
```

Такие окружения полезны, когда нужно:

- показать несколько связанных уравнений;
- разбить длинную формулу на строки;
- сделать запись более читаемой.

---

# 3.6 Жирный шрифт в формулах

Для выделения символов в формулах можно использовать пакет `bm`.

```latex
\usepackage{bm}

$(x+\bm{y})(x-\bm{y}) = x^2 - \bm{y}^2$
```

Это позволяет:

- выделять важные переменные;
- оформлять векторы и параметры;
- делать части формулы визуально заметнее.

---

# 3.7 Пакет `mathtools`

`mathtools` расширяет возможности `amsmath`.

Пример выравнивания элементов матрицы по правому краю:

```latex
\[
\begin{pmatrix*}[r]
10 & 11 \\
1 & 2 \\
-5 & -6
\end{pmatrix*}
\]
```

Пакет полезен для более точного контроля оформления математических объектов.

---

# 3.8 Unicode Math

Пакет `unicode-math` позволяет использовать современные OpenType-шрифты для математики.

```latex
\usepackage{unicode-math}
\setmainfont{TeX Gyre Pagella}
\setmathfont{TeX Gyre Pagella Math}

\[
\log \alpha + \log \beta = \log(\alpha\beta)
\]
```

Это особенно удобно при работе с XeLaTeX и LuaLaTeX.
