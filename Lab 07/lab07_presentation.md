# 3. Презентации с Beamer

Beamer позволяет создавать слайды с темами и структурой презентации.

```latex
\documentclass{beamer}
\usetheme{Copenhagen}
\author{Bert}
\title{A tale of two primes}
\begin{document}
\begin{frame}
\titlepage
\end{frame}
\end{document}
```

Это базовый шаблон презентации LaTeX.

---

# 3.1 Структура презентации

Для каждого слайда используется окружение `frame`.

```latex
\begin{frame}[t]{Article}
Some text about the article.
\end{frame}
```

Ключевые элементы:
- титульный слайд;
- обычные слайды;
- заголовки и подзаголовки;
- темы оформления.

---

# 3.2 Паузы и появление элементов

Beamer поддерживает динамическое раскрытие контента.

```latex
\pause
\uncover<2->{Second block appears later}
```

Это полезно для:
- пошагового объяснения;
- удержания внимания;
- раскрытия определения и теоремы по очереди.

---

# 3.3 Оформление презентаций

Внешний вид можно менять через темы:

```latex
\usetheme{Warsaw}
\usecolortheme{beaver}
```

Советы для коротких презентаций из отчёта:
- выделять главную тему сразу;
- использовать изображения вместо длинных текстов;
- давать примеры вместо абстрактных определений;
- следить за временем.

# 3.4 Постеры в LaTeX

Рассматривались три основных способа:

1. `a0poster`
2. `beamerposter`
3. `tikzposter`

Каждый подход подходит для научных постеров, но отличается уровнем гибкости и стилем оформления.

---

# 3.5 Beamerposter

`beamerposter` использует возможности Beamer для оформления научных постеров.

Преимущества:
- единый стиль с презентациями;
- блоковая структура;
- удобная работа с колонками, `block`, `exampleblock`, `alertblock`.

```latex
\usepackage[orientation=portrait,size=a0,scale=1]{beamerposter}
```

---

# 3.6 Tikzposter

`tikzposter` удобен для полностью управляемой постерной верстки.

```latex
\documentclass[24pt, a0paper, portrait]{tikzposter}
\usetheme{Default}
```

Он полезен, когда нужно совмещать постерные блоки, TikZ-графику и кастомное размещение элементов.
